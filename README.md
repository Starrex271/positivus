# Positivus

A modern and responsive digital marketing website built with HTML, CSS, and JavaScript. The project focuses on clean UI/UX design, responsiveness across devices, and smooth user interaction.

---

## 🚀 Features

* Responsive modern UI design
* Interactive navigation bar
* Smooth scrolling effects
* Mobile-friendly layout
* Clean and structured codebase
* Reusable components and sections
* Fast and lightweight performance

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and responsiveness
* **JavaScript** – Interactivity and dynamic behavior

---

## ⚡ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/starrex271/positivus.git
```

### 2. Navigate into the Project Folder

```bash
cd positivus
```

### 3. Open in Browser

Simply open `home.html` in your browser.
